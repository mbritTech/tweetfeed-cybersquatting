# Tweetfeed Cybersquatting
An easy filtering system for **cybersquatting detection**.

This system is composed by the filtering process of the
**.es domains** with a list of **suspicious phishing domains**.

## Sources
- Spanish domains list - **[Dominios.es](https://www.dominios.es/es/sobre-dominios/estadisticas)**
- Suspicious phishing domains list - **[Tweetfeed](https://tweetfeed.live/)**

## Required Libraries
`pip install request` 

`pip install tldextract`

`pip install csv`

## Execution
Navigate to the **[main file](main.py)** and run it.
